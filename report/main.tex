\section*{Team Contribution}

This project was completed collaboratively by Yifan Luo and Ming Cheng. Both authors contributed to project discussion, experimental design, implementation, result analysis, and report writing.

\begin{abstract}
This report presents three representative computer vision tasks based on deep learning, including fine-grained image classification, object detection and multi-object tracking, and semantic segmentation. For image classification, pretrained convolutional neural networks and attention-based architectures are explored on the Oxford-IIIT Pet Dataset. For object detection and tracking, YOLOv8 and ByteTrack are applied to the VisDrone dataset and real-world traffic videos. For semantic segmentation, a U-Net model is implemented from scratch and trained with different loss function configurations. Experimental results demonstrate the effectiveness of transfer learning, modern detection and tracking frameworks, and dice loss function across different visual understanding tasks.
\end{abstract}
