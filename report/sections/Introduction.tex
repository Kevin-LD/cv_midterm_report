\section{Introduction}

This midterm project investigates several fundamental deep learning based computer vision tasks, including image classification, object detection and multi-object tracking, and semantic segmentation. The project aims to provide practical experience in model training, transfer learning, loss function engineering, and performance evaluation under different visual understanding settings.

In Task 1, pretrained convolutional neural networks and attention-based architectures are explored for fine-grained pet classification on the Oxford-IIIT Pet Dataset. In Task 2, a YOLOv8-based detector is fine-tuned on the VisDrone dataset and further integrated with a multi-object tracking framework for video analysis. In Task 3, a U-Net segmentation model is implemented from scratch, and different loss function configurations are compared for pixel-level semantic segmentation.

Throughout the project, experiments are conducted with comprehensive analysis of training dynamics, quantitative performance metrics, and qualitative visualization results.
