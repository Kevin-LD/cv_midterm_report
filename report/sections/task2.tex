\section{Task 2: Scene Object Detection and Multi-Object Tracking}

This task focuses on scene object detection and video multi-object tracking in complex urban environments. The objective is not only to detect objects in individual frames, but also to maintain consistent object identities across time and further support downstream video analysis tasks such as occlusion analysis and line-crossing counting.

To accomplish this task, YOLOv8 is fine-tuned on the VisDrone2019-DET dataset for small-object detection in dense aerial-style scenes. Two experimental configurations with different model scales and input resolutions are trained and evaluated. The trained detector is then integrated with the ByteTrack tracking algorithm for video multi-object tracking on a real-world traffic scene. Based on the tracking results, occlusion recovery and ID switching behaviors are analyzed under challenging scenarios involving pedestrian interaction and environmental occlusion. Finally, a line-crossing counting module is implemented using object trajectories and geometric intersection tests to estimate vehicle flow through a selected roadway exit.
