\subsection{Summary}
In this task, we investigate semantic segmentation on the Oxford-IIIT Pet Dataset using a fully customized U-Net architecture. A complete segmentation pipeline is developed from scratch, including dataset preprocessing, mask transformation, encoder-decoder network construction, skip connection design, loss function exploration, optimization strategies, training monitoring, and quantitative evaluation.

To improve segmentation performance, multiple loss functions are explored, including Cross-Entropy Loss, Dice Loss, and a combined hybrid loss formulation. Demonstrating that the combined loss achieves better performance compared with using either loss individually.

Finally, analysis of the class-wise confusion matrix and examination of misclassified regions provide insights into the remaining segmentation challenges, such as boundary ambiguities between similar pet parts and occasional mislabeling in underrepresented classes. These analyses help interpret model behavior and guide potential avenues for future improvements in semantic segmentation performance.
